WINMERGE

WinMerge este un utilitar cu Sursă Deschisă de comparare şi fuziune 
de fişiere care rulează pe toate versiunile moderne ale sistemului
de operare Windows. Unele funcţionalităţi necesită instalarea de
componente adiţionale.

Ultima versiune de WinMerge şi alte informaţii despre WinMerge sunt
disponibile la
https://winmerge.org

Start rapid către utilizarea WinMerge:
Citiţi capitolul Quick-start din manualul on-line pentru a vă
familiariza cu WinMerge:
https://manual.winmerge.org/QuickStart.html

Manualul HTML:
Manualul este disponibil la
https://manual.winmerge.org/
este posibil să-l aveţi instalat (dacă s-a optat pentru aceasta) local,
dar este şi copiabil separat de la https://winmerge.org/
(vezi documentaţia)

Suport arhivare:
WinMerge foloseşte 7-Zip pentru a suporta accesul la arhive de fişiere.
7-Zip (https://www.7-zip.org) este o unealtă de arhivare cu Sursă Deschisă.
Pentru a instala suportul de arhivare, copiaţi kit-ul plug-in-ului 7-Zip
de la https://winmerge.org/ .

Instalarea suportului de arhivare:
Este recomandat să instalaţi aplicaţia 7-Zip. Dacă acest lucru nu poate fi
făcut din oarecare motiv, instalatorul plug-in-ului 7-Zip poate instala
doar fişierele necesare pentru activarea suportului de instalare. Luaţi notă
că acest lucru NU va permite funcţionarea de sine stătătoare a lui 7-Zip, ci
doar suportul de arhivare pentru WinMerge.

Suportul pentru scripturi:
Dacă doriţi să lucraţi cu scripturi aveţi nevoie ca Windows Scripting Host
să fie instalat. Dacă observaţi erori care au ca sursă scripturile, vizitaţi
https://msdn.microsoft.com/library/default.asp?url=/downloads/list/webdev.asp
şi verificaţi daca Windows Scripting Host este la zi şi funcţionabil.

Defecte şi solicitarea de noi funcţionalităţi:
Defectele şi solicitarea de noi funcţionalităţi trebuie înregistrate pe
listele de Defecte (Bugs) şi Funcţionalităţi (RFE) de la sourceforge.net.

Lista de defecte:
https://sourceforge.net/tracker/?group_id=13216&atid=113216
Atunci când înregistraţi un defect, vă rugăm să ne spuneţi versiunea de
WinMerge pe care o folosiţi! WinMerge versiunea 2.2.0 sau mai noi pot
scrie configurarea sub care rulează într-un fişier.
Această comandă este accesibilă din meniul Ajutor/Configurare.
Vă rugăm să ataşaţi şi această informaţie (ca un ataşament) la raportarea
dumneavoastră, conţine informaţii pentru dezvoltatori.
